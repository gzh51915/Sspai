## 目录说明
- 前端
    - node_modules：项目包文件夹
    - public：公共资源
    - src：源码
        - components：组件
        - pages：页面
        - store：redux仓库
        - utils：工具文件夹

- 后端
    - src：源码
        - api: 拦截请求处理文件
        - config：接口地址配置
        - component：工具文件夹
        - page：页面
